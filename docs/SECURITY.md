# Security Policy

## Supported Versions

Currently, we are supporting the following versions with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 7.3.x   | :white_check_mark: |

Support for other versions is available with [subscription plans](https://www.membrane-api.io/api-gateway-pricing.html).

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, use GitHub's private vulnerability reporting for this repository: go to the
[Security tab](https://github.com/membrane/api-gateway/security) and click
"Report a vulnerability", or open
[a new advisory directly](https://github.com/membrane/api-gateway/security/advisories/new).
Alternatively, send an email to info@predic8.de. We will look at your report immediately
and evaluate it. Do not send confidential information in plain text; encrypt the email body
using the GPG key at https://keys.openpgp.org/vks/v1/by-fingerprint/2FB0F3ED57EF0A8A9CE847C18A006E355B8A65F6.
